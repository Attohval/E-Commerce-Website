AuroraShop — Demo E‑Commerce Template

Files:
- index.html — Home page
- shop.html — Shop with filters
- product.html — Product details
- categories.html — Category listings
- cart.html — Cart UI
- wishlist.html — Wishlist
- checkout.html — Checkout form
- login.html / register.html — Authentication UIs
- about.html / contact.html / faq.html — Info pages
- style.css — Main stylesheet
- script.js — Global interactions
- aurora-api.js — Small client-side API for demo products

How to run:
Open `index.html` in a browser (or serve the folder with a static server).

Notes:
- This is a frontend-only demo. Replace demo product data and connect to an API/backend for production.
- Images use Unsplash links for placeholders.
- The UI is responsive and includes dark mode, hero slider, cart/wishlist using `localStorage`.
